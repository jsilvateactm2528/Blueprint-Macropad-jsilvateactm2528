# Blueprint-Macropad-jsilvateactm2528
João Macropad is a 7 key macropad with 2 rotary encoders, an OLED Display. It also has 7 SK6812 MINI-E LEDs, and uses QMK firmware 

I spend a lot of hours on this build (i forgot to count the time). João Macropad is a compact macropad featuring 7 mechanical keys, 2 rotary encoders, a OLED display, and 7 rgb leds.
